# Using alternative/faster linker zld
1. Install zld
    brew install michaeleisel/zld/zld   
2. Modify cargo file
vim ~/.cargo/config

```
[target.x86_64-apple-darwin]
rustflags = ["-C", "link-arg=-fuse-ld=/usr/local/bin/zld"]
```
